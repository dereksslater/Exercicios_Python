# 🐍 Exercícios de Lógica em Python — Aprendizado e Insights

Este repositório foi criado para centralizar meus estudos de **Lógica de Programação**, utilizando **Python** como linguagem base. O foco aqui não é construir projetos complexos, mas sim treinar a base de tudo: algoritmos, raciocínio lógico e resolução de problemas.

Os exercícios são baseados nas aulas do **Prof. Elder (UFSC)**, do canal [Prof. Elder no YouTube](https://www.youtube.com/@ProfElder).

---

## 🎯 Por que estou fazendo estes exercícios?

* **Fortalecer a base:** A sintaxe muda de uma linguagem para outra, mas a lógica por trás de um bom código é universal. 
* **Pensamento Analítico:** Aprender a quebrar um problema grande em partes menores antes de começar a digitar o código.
* **Agilidade de Desenvolvimento:** Dominar estruturas condicionais, laços de repetição e manipulação de dados para escrever códigos mais limpos e eficientes.

---

## 🧠 Dicas do Curso & Conceitos Chave

Este espaço serve como um "mural de insights" sobre o que estou praticando e aplicando nos arquivos deste repositório:

* **Entrada e Saída de Dados:** Dominar o fluxo de como o programa recebe informações do usuário e processa a resposta.
* **Estruturas Condicionais (`if/elif/else`):** O coração da tomada de decisão. A dica aqui é sempre testar os cenários de borda (extremos) para garantir que o código não quebre.
* **Laços de Repetição (`while/for`):** Automatizar processos. Entender a diferença de quando usar um laço com parada definida (`for`) versus uma condição dinâmica (`while`).
* **Estruturas de Dados:** O uso inteligente de listas para armazenar e organizar informações de forma eficiente.
